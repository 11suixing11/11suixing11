### 💡 我相信的事

- **简单是特性，不是妥协。** 没有服务器，没有账号，没有追踪。
- **数据归你。** 一切都在浏览器的 IndexedDB 里。
- **设计应该有温度。** 不是每个应用都要看起来像飞船。有时候你只想要一张老纸和一道柔影。
