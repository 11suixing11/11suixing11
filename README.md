### 🚀 正在做的事

**[MindNotes Pro](https://github.com/11suixing11/mindnotes-pro)** — 暖色纸纹画布笔记本

3 个依赖。0 个 CDN。纯本地运行。打开就能画，能写，能存。不联网，不注册，不打扰。

> 每个白板应用都想让我注册账号、同步到云端。我只是想要一块画布。

[**在线使用 →**](https://11suixing11.github.io/mindnotes-pro/)

### 🛠️ 技术选择

```text
React 18  ·  TypeScript 5  ·  Vite 5  ·  Zustand  ·  Canvas API
```

3 个生产依赖。不是偷懒，是信仰。

---

### 💡 我相信的事

- **简单是特性，不是妥协。** 3 个依赖和 0 个 CDN 不是限制，是设计选择。
- **数据归你。** 没有服务器，没有账号，没有追踪。一切都在浏览器的 IndexedDB 里。关掉标签页，明天再来，一切都在。
- **设计应该有温度。** 不是每个应用都要看起来像飞船。有时候你只想要一张老纸和一道柔影。
